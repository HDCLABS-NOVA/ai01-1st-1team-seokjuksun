## 💡주요 기능
---
 - 공정 데이터 실시간 예측
 - 오류 발생 시 알림 시스템
 - 기온, 습도 정보 확인
 - 생산량 정보 확인

<br>

## 🛠기술 스택
---
 - **Python 3.13**
 - **데이터 수집**: 기상청 API 활용
 - **분석**: Pandas, Numpy
 - **시각화**: Matplotlib, Seaborn, Ploty
 - **모델링**: XGBoost
